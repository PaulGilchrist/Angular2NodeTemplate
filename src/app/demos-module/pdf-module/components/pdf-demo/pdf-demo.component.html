PDF Generation Demo

This button demonstrates dynamic creation of a PDF document supporting headers, footers, page breaks, images (base64 only), tables, columns, lists, backgrounds, margins, orientation, etc. PDF generation has been tested on IE, Edge, Chrome, Firefox, and iOS. Safari and Android testing are still needed.

Modal Generation Demo

This button demonstrates dynamic creation of a Modal dialog box using standard Bootstrap control. Additional inputs and outputs or service connections can be used with this model as needed for form support or other more advanced usage.
